StellarisWorkshop
StellarisWorkshop is a tech-oriented skyblock modpack built around Ex Nihilo, Create, GregTech, and Mekanism, developed using the GregTech Modern Community Pack (GCPM) template. It integrates Botania and Magical Crops to support resource production, offering a complete progression with detailed quests and refined tweaks.

The pack features a full-stage Ex Nihilo resource system enhanced with custom additions like the GregTech Sieve and Create Sieve, enabling endless resource generation from your base. Automation is streamlined with the latest AE2 and popular add-ons (EAEP, AECS). Content is expanded through numerous GregTech multiblocks (from GTNN, GTCA, GTSW) and classic GregTech elements. Quality-of-life improvements include wireless power, programmable hatches, advanced terminals, giant buffers, larger pattern terminals, and other originally private features now publicly available.
